## Guildedの始め方

[Guildedのwebサイト](https://www.guilded.gg/)の中の「Sign up」を押すと、出てくる「Get atarted with Guilded」の中の「Username」には使用したい名前（日本語や絵文字では登録不可）「Email」には登録するアカウントのメールアドレス、Passwordには使用するパスワードを登録する事で始められる。
